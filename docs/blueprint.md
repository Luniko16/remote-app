# **App Name**: ResumAI

## Core Features:

- AI Content Generation: Automatically generate resume content, including summaries and job descriptions, based on user inputs using an AI tool.
- Smart Optimization: Generate industry-specific keywords and phrases. Analyze job descriptions to tailor resume content and ensure ATS compatibility, using an AI tool.
- Multiple Export Options: Export completed resumes in PDF, DOCX, and HTML formats.
- Professional Templates: Provide at least three distinct visual resume templates with customizable colors, fonts, and layout elements.
- Intelligent Assistance: Offer smart suggestions and real-time content recommendations using an AI tool. Include tips and guidance for resume best practices.
- User Interface: Clean, responsive design with a real-time preview and step-by-step resume building process.
- User Data Management: Implement local storage for user data privacy and security.

## Style Guidelines:

- Primary color: A muted teal (#73A7A7) evoking a sense of calm professionalism, inspired by the reliable and trustworthy aspects of career development.
- Background color: A very light off-white (#F2F2F2), almost the same hue as the primary, offering a clean and unobtrusive backdrop.
- Accent color: A deep cyan (#307A7A), an analogous hue, provides a contrasting highlight without disrupting the overall color scheme.
- Headline font: 'Belleza' (sans-serif) for headlines, adding personality aligned to design.
- Body font: 'Alegreya' (serif) will pair with Belleza, to facilitate readability of longer text passages
- Ensure a clean, structured layout that is easy to read and ATS-friendly.
- Use professional, minimalistic icons to represent different sections of the resume (e.g., work experience, education, skills).
- Subtle animations to enhance user experience, such as smooth transitions between steps or a progress bar while generating content.
